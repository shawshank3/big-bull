# Big Bull UI Starter (Vite + React + RTK Query)

This repository is the UI for the Big Bull project. It uses Vite + React and Redux Toolkit Query for API fetching.

Quick start (client only):

1. Install dependencies:

```bash
npm install
```

2. Start the Vite dev server:

```bash
npm run dev
```

The client runs on `http://localhost:5173` by default. Configure any backend separately and ensure API endpoints are reachable (the client proxies `/api` to `http://localhost:4000` by default in `vite.config.js`).
