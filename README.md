# big-bull-api
Mutual Funds - Stocks Backend

Simple API with holdings endpoints used by the UI.

Quick start:

```bash
cd e:\\Project\\MERN Project\\big-bull-api
npm install
cp .env.example .env
npm run dev
```

Endpoints:

- `GET /holdings/mutuals` — list of mutual fund holdings
- `GET /holdings/stocks` — list of stock holdings
